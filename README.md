# granularity lens 

inspired by [Dense X Retrieval: What Retrieval Granularity Should We Use?](https://arxiv.org/abs/2312.06648)

## setup
1. backend:
```bash
cd backend
pip install -r requirements.txt
python -m uvicorn main:app --reload 
```

2. frontend:
```bash
cd frontend
npm install
npm run dev
```
